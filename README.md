# Conversor de Unidades de Temperatura

Este é um projeto de software web colaborativo que realiza a conversão de temperaturas entre Celsius, Fahrenheit e Kelvin. O projeto foi desenvolvido puramente no front-end, sem dependências de execução de código no servidor.

## 📌 Histórico de Versões

### Versão v1.0.0 (Atual)
* **Foco:** Funcionalidade e Estrutura Base.
* **Descrição:** Interface HTML estruturada com todos os elementos necessários para a conversão (inputs, seletores e botão) e estilização CSS básica e simplificada para fins funcionais.
